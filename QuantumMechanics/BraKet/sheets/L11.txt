```latex
\section{Density operators: pure and mixed}
A pure state is $|\psi\rangle$ with density operator
\[
\rho=|\psi\rangle\langle\psi|.
\]
A mixed state (ensemble) is
\[
\rho=\sum_i p_i |\psi_i\rangle\langle\psi_i|,\quad p_i\ge0,\ \sum_i p_i=1.
\]
Different ensembles can produce the same $\rho$ (non-uniqueness of decomposition).

\begin{warnbox}
Mixture is not superposition: $\rho=\frac12(|0\rangle\langle0|+|1\rangle\langle1|)$ is not the same as
$|\!+\rangle\langle+\!|$ even though both involve $|0\rangle,|1\rangle$.
\end{warnbox}

\section{Trace rule and purity}
Expectation values:
\[
\langle A\rangle=\Tr(\rho A).
\]
Purity:
\[
\gamma=\Tr(\rho^2),\quad \gamma=1 \text{ for pure states, } \gamma<1 \text{ for mixed states}.
\]

\subsection{Bloch form (qubit)}
Any qubit state can be written
\[
\rho=\frac12(I+\mathbf r\cdot\sigma),\quad |\mathbf r|\le 1.
\]
Then $\Tr(\rho^2)=\frac12(1+|\mathbf r|^2)$.

\section{Partial trace and reduced states}
For bipartite systems,
\[
\rho_A=\Tr_B(\rho_{AB}).
\]
Key property:
\[
\Tr(\rho_{AB}(A\otimes I))=\Tr(\rho_A A).
\]
Operational meaning: $\rho_A$ captures all statistics of measurements performed only on subsystem $A$.

\section{Quantum channels: CPTP maps}
Open-system dynamics and measurements (when outcomes are ignored) are described by linear maps
\[
\mathcal E:\rho\mapsto \mathcal E(\rho)
\]
that are:
\begin{itemize}
\item Completely positive (CP)
\item Trace preserving (TP)
\end{itemize}
Such maps are called CPTP.

\subsection{Kraus representation}
A CPTP map admits an operator-sum form
\[
\mathcal E(\rho)=\sum_k M_k\rho M_k^\dagger,
\quad \sum_k M_k^\dagger M_k = I.
\]
If outcomes $k$ are observed (instrument), then
\[
p_k=\Tr(M_k\rho M_k^\dagger),\qquad \rho_k=\frac{M_k\rho M_k^\dagger}{p_k}.
\]

\section{Choi matrix preview}
Define the (unnormalized) maximally entangled ket $|\Omega\rangle=\sum_{j=1}^d |j\rangle\otimes|j\rangle$.
The Choi matrix of $\mathcal E$ is
\[
J(\mathcal E) = (\mathcal E\otimes I)(|\Omega\rangle\langle\Omega|).
\]
Key facts (preview):
\begin{itemize}
\item $\mathcal E$ is CP iff $J(\mathcal E)\ge 0$.
\item $\mathcal E$ is TP iff $\Tr_{\text{out}}(J(\mathcal E))=I_{\text{in}}$ (index conventions vary; we fix below in exercises).
\end{itemize}

\begin{mapbox}
\begin{tabular}{@{}ll@{}}
\toprule
\textbf{Object} & \textbf{Condition / formula}\\ \midrule
State $\rho$ & $\rho\ge0$, $\Tr\rho=1$\\
Reduced state & $\rho_A=\Tr_B\rho_{AB}$\\
Channel (CPTP) & $\mathcal E(\rho)=\sum_k M_k\rho M_k^\dagger$, $\sum_k M_k^\dagger M_k=I$\\
Instrument & $p_k=\Tr(M_k\rho M_k^\dagger)$, $\rho_k=M_k\rho M_k^\dagger/p_k$\\
Choi matrix & $J(\mathcal E)=(\mathcal E\otimes I)(|\Omega\rangle\langle\Omega|)$\\
CP test & $J(\mathcal E)\ge0$\\
TP test & partial trace condition on $J(\mathcal E)$\\
\bottomrule
\end{tabular}
\end{mapbox}

For Choi preview, keep it computational in $d=2$ with a simple channel (depolarizing or amplitude damping).
```

## **S1 — Pure-state density matrix and idempotence**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath,amssymb,physics}\geometry{margin=1in}\begin{document}
\section*{L11-S1 Pure state density matrix — Problem}
Let $|\psi\rangle=\alpha|0\rangle+\beta|1\rangle$ with $|\alpha|^2+|\beta|^2=1$.
Compute $\rho=|\psi\rangle\langle\psi|$ and show $\rho^2=\rho$.

\newpage\section*{Suggestions}\begin{itemize}
\item Write $\rho$ as a $2\times2$ matrix using outer product.
\item Multiply $\rho$ by itself.
\end{itemize}

\newpage\section*{Advanced suggestions}\begin{itemize}
\item Use $|\psi\rangle\langle\psi|\psi\rangle\langle\psi| = |\psi\rangle(\langle\psi|\psi\rangle)\langle\psi|$.
\end{itemize}

\newpage\section*{Detailed solution}
\[
\rho=\begin{pmatrix}\alpha\\ \beta\end{pmatrix}(\alpha^*,\beta^*)
=\begin{pmatrix}|\alpha|^2 & \alpha\beta^*\\ \beta\alpha^* & |\beta|^2\end{pmatrix}.
\]
Then
\[
\rho^2=(|\psi\rangle\langle\psi|)(|\psi\rangle\langle\psi|)
=|\psi\rangle(\langle\psi|\psi\rangle)\langle\psi|=|\psi\rangle\langle\psi|=\rho,
\]
since $\langle\psi|\psi\rangle=1$.
\end{document}
```

## **S2 — Mixed state and purity**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath,amssymb,physics}\geometry{margin=1in}\begin{document}
\section*{L11-S2 Mixture purity — Problem}
Let $\rho=p|0\rangle\langle0|+(1-p)|1\rangle\langle1|$ with $0\le p\le1$.
Compute $\Tr(\rho^2)$ and interpret.

\newpage\section*{Suggestions}\begin{itemize}
\item $\rho=\mathrm{diag}(p,1-p)$.
\item Square and take trace.
\end{itemize}

\newpage\section*{Advanced suggestions}\begin{itemize}
\item Show purity is $1$ only at $p=0$ or $1$ and minimal at $p=1/2$.
\end{itemize}

\newpage\section*{Detailed solution}
$\rho=\begin{pmatrix}p&0\\0&1-p\end{pmatrix}$ so
$\rho^2=\mathrm{diag}(p^2,(1-p)^2)$ and
\[
\Tr(\rho^2)=p^2+(1-p)^2=1-2p(1-p).
\]
This equals $1$ only for $p\in\{0,1\}$ (pure). It is minimal at $p=1/2$ giving $\Tr(\rho^2)=1/2$ (maximally mixed qubit).
\end{document}
```

## **S3 — Trace rule computation**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath,amssymb}\geometry{margin=1in}\begin{document}
\section*{L11-S3 Trace rule — Problem}
Let $\rho=\begin{pmatrix}1/2&1/2\\1/2&1/2\end{pmatrix}$ and $A=\sigma_z=\begin{pmatrix}1&0\\0&-1\end{pmatrix}$.
Compute $\langle A\rangle=\Tr(\rho A)$.

\newpage\section*{Suggestions}\begin{itemize}
\item Multiply $\rho A$ then take trace.
\end{itemize}

\newpage\section*{Advanced suggestions}\begin{itemize}
\item Recognize $\rho=|+\rangle\langle+|$ and reason physically.
\end{itemize}

\newpage\section*{Detailed solution}
\[
\rho A=\begin{pmatrix}1/2&1/2\\1/2&1/2\end{pmatrix}\begin{pmatrix}1&0\\0&-1\end{pmatrix}
=\begin{pmatrix}1/2&-1/2\\1/2&-1/2\end{pmatrix}.
\]
Trace is $1/2+(-1/2)=0$. Thus $\langle A\rangle=0$.
\end{document}
```

## **S4 — Bloch vector ↔ density matrix**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath,amssymb}\geometry{margin=1in}\begin{document}
\section*{L11-S4 Bloch representation — Problem}
Let $\rho=\frac12(I+r_z\sigma_z)$. Write $\rho$ explicitly as a matrix and compute $\Tr(\rho^2)$.

\newpage\section*{Suggestions}\begin{itemize}
\item $\sigma_z=\mathrm{diag}(1,-1)$.
\item Use $\Tr(\rho^2)=\frac12(1+r_z^2)$ for qubits or compute directly.
\end{itemize}

\newpage\section*{Advanced suggestions}\begin{itemize}
\item Check pure-state condition: $|r_z|=1$.
\end{itemize}

\newpage\section*{Detailed solution}
\[
\rho=\frac12\left(\begin{pmatrix}1&0\\0&1\end{pmatrix}+r_z\begin{pmatrix}1&0\\0&-1\end{pmatrix}\right)
=\begin{pmatrix}\frac{1+r_z}{2}&0\\0&\frac{1-r_z}{2}\end{pmatrix}.
\]
Then
\[
\Tr(\rho^2)=\left(\frac{1+r_z}{2}\right)^2+\left(\frac{1-r_z}{2}\right)^2=\frac12(1+r_z^2).
\]
\end{document}
```

## **S5 — Partial trace for a 2‑qubit pure state**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath,amssymb,physics}\geometry{margin=1in}\begin{document}
\section*{L11-S5 Partial trace — Problem}
Let $|\Psi\rangle=\sqrt{0.8}|00\rangle+\sqrt{0.2}|11\rangle$ and $\rho_{AB}=|\Psi\rangle\langle\Psi|$.
Compute $\rho_A=\Tr_B(\rho_{AB})$.

\newpage\section*{Suggestions}\begin{itemize}
\item Expand $\rho_{AB}$ into four terms and trace over $B$.
\item Cross terms vanish due to $\langle0|1\rangle=0$.
\end{itemize}

\newpage\section*{Advanced suggestions}\begin{itemize}
\item Recognize it is already in Schmidt form; reduced eigenvalues are $\{0.8,0.2\}$.
\end{itemize}

\newpage\section*{Detailed solution}
\[
\rho_{AB}=0.8|00\rangle\langle00|+\sqrt{0.16}|00\rangle\langle11|+\sqrt{0.16}|11\rangle\langle00|+0.2|11\rangle\langle11|.
\]
Trace over $B$:
\[
\Tr_B(|00\rangle\langle00|)=|0\rangle\langle0|,\quad
\Tr_B(|11\rangle\langle11|)=|1\rangle\langle1|,
\]
cross terms vanish. Hence
\[
\rho_A=0.8|0\rangle\langle0|+0.2|1\rangle\langle1|=\begin{pmatrix}0.8&0\\0&0.2\end{pmatrix}.
\]
\end{document}
```

## **S6 — Verify local expectation identity**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath,amssymb,physics}\geometry{margin=1in}\begin{document}
\section*{L11-S6 Local statistics identity — Problem}
For $\rho_{AB}$ from S5 and $A=\sigma_z$, verify $\Tr(\rho_{AB}(A\otimes I))=\Tr(\rho_A A)$.

\newpage\section*{Suggestions}\begin{itemize}
\item Use $\rho_A$ from S5.
\item Compute RHS directly; interpret LHS as expectation of $\sigma_z$ on subsystem $A$.
\end{itemize}

\newpage\section*{Advanced suggestions}\begin{itemize}
\item LHS can be computed by weighted outcomes: $|0\rangle$ occurs with prob 0.8 and $|1\rangle$ with prob 0.2.
\end{itemize}

\newpage\section*{Detailed solution}
RHS:
\[
\Tr(\rho_A\sigma_z)=0.8(1)+0.2(-1)=0.6.
\]
LHS: measurement of $\sigma_z$ on $A$ yields +1 with probability 0.8 and -1 with probability 0.2, so expectation is also 0.6. Hence equality holds.
\end{document}
```

## **S7 — Kraus completeness check**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath,amssymb}\geometry{margin=1in}\begin{document}
```

## **C1 — Trace rule equivalence**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath,amssymb,physics}\geometry{margin=1in}\begin{document}
\section*{L11-C1 Trace rule equivalence — Problem}
Prove $\langle\psi|A|\psi\rangle=\Tr(|\psi\rangle\langle\psi|A)$ and extend to mixtures $\rho=\sum_i p_i|\psi_i\rangle\langle\psi_i|$.

\newpage\section*{Suggestions}\begin{itemize}
\item Use an orthonormal basis and compute trace as sum of diagonal entries.
\item Use cyclicity of trace.
\end{itemize}

\newpage\section*{Advanced suggestions}\begin{itemize}
\item Show $\Tr(|\psi\rangle\langle\psi|A)=\Tr(A|\psi\rangle\langle\psi|)=\langle\psi|A|\psi\rangle$.
\end{itemize}

\newpage\section*{Detailed solution}
Using cyclicity, $\Tr(|\psi\rangle\langle\psi|A)=\Tr(A|\psi\rangle\langle\psi|)$.
In any basis, $A|\psi\rangle\langle\psi|$ has trace equal to $\langle\psi|A|\psi\rangle$ (standard identity).
For mixtures, linearity of trace gives
\[
\Tr(\rho A)=\sum_i p_i\Tr(|\psi_i\rangle\langle\psi_i|A)=\sum_i p_i\langle\psi_i|A|\psi_i\rangle.
\]
\end{document}
```

## **C2 — Partial trace basis-independence (formal)**

```latex
\documentclass[11pt]{article}\usepackage{geometry}\geometry{margin=1in}\begin{document}
\section*{L11-C2 Basis-independence — Problem}
Give a formal argument that $\rho_A=\Tr_B(\rho_{AB})$ is independent of the orthonormal basis used to trace over $B$.

\newpage\section*{Suggestions}
\begin{itemize}
\item Use that different ONBs are related by a unitary.
\item Use invariance of trace under unitary similarity on subsystem $B$.
\end{itemize}

\newpage\section*{Advanced suggestions}
\begin{itemize}
\item Argue via defining property: $\Tr(\rho_{AB}(A\otimes I))=\Tr(\rho_A A)$ uniquely determines $\rho_A$.
\end{itemize}

\newpage\section*{Detailed solution}
One argument: $\rho_A$ is uniquely defined by the identity $\Tr(\rho_{AB}(A\otimes I))=\Tr(\rho_A A)$ for all $A$.
Since the left side is basis-free, $\rho_A$ is basis-free.
Equivalently, switching the trace basis by a unitary on $B$ leaves the trace unchanged.
```

## **C3 — Kraus completeness ⇒ CPTP (TP + CP sketch)**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath,amssymb,physics}\geometry{margin=1in}\begin{document}
\section*{L11-C3 Kraus completeness implies CPTP — Problem}
Let $\mathcal E(\rho)=\sum_k M_k\rho M_k^\dagger$ with $\sum_k M_k^\dagger M_k=I$.
Show $\Tr(\mathcal E(\rho))=\Tr(\rho)$ and give a short argument why $\mathcal E$ is completely positive.

\newpage\section*{Suggestions}\begin{itemize}
\item Use cyclicity: $\Tr(M\rho M^\dagger)=\Tr(\rho M^\dagger M)$.
\item For CP: argue positivity of $\sum_k (M_k\otimes I) X (M_k^\dagger\otimes I)$ when $X\ge0$.
\end{itemize}

\newpage\section*{Advanced suggestions}\begin{itemize}
\item Mention Stinespring dilation as an alternative CP argument.
\end{itemize}

\newpage\section*{Detailed solution}
Trace preservation:
\[
\Tr(\mathcal E(\rho))=\sum_k \Tr(M_k\rho M_k^\dagger)=\sum_k \Tr(\rho M_k^\dagger M_k)
=\Tr\left(\rho\sum_k M_k^\dagger M_k\right)=\Tr(\rho).
\]
Complete positivity: for any ancilla space, define $(\mathcal E\otimes I)(X)=\sum_k (M_k\otimes I)X(M_k^\dagger\otimes I)$.
If $X\ge0$ then each term is positive and the sum is positive, so $\mathcal E$ is CP.
\end{document}
```

## **C5 — Kraus extraction from Choi (preview)**

```latex
\documentclass[11pt]{article}\usepackage{geometry,amsmath}\geometry{margin=1in}\begin{document}
\section*{L11-C5 Kraus from Choi (preview) — Problem}
Outline how to obtain Kraus operators from a Choi matrix $J(\mathcal E)$:
(i) diagonalize $J=\sum_\ell \lambda_\ell |v_\ell\rangle\langle v_\ell|$ with $\lambda_\ell\ge0$,
(ii) reshape each $|v_\ell\rangle$ into a matrix $V_\ell$,
(iii) set $M_\ell=\sqrt{\lambda_\ell}\,V_\ell$.

\newpage\section*{Suggestions}
\begin{itemize}
\item Recall vectorization: $\mathrm{vec}(M)=\sum_{ij} M_{ij}|i\rangle|j\rangle$.
\item $|\Omega\rangle=\sum_i |i\rangle|i\rangle$ relates vec to action on half of maximally entangled state.
\end{itemize}

\newpage\section*{Advanced suggestions}
\begin{itemize}
\item Verify that $J=\sum_\ell \mathrm{vec}(M_\ell)\mathrm{vec}(M_\ell)^\dagger$.
\item Show this implies $\mathcal E(\rho)=\sum_\ell M_\ell\rho M_\ell^\dagger$.
\end{itemize}

\newpage\section*{Detailed solution (model outline)}
Since $J\ge0$, diagonalize it: $J=\sum_\ell \lambda_\ell |v_\ell\rangle\langle v_\ell|$ with $\lambda_\ell\ge0$.
Define $|w_\ell\rangle=\sqrt{\lambda_\ell}|v_\ell\rangle$. Each $|w_\ell\rangle$ can be reshaped into a matrix $M_\ell$ via $\mathrm{vec}(M_\ell)=|w_\ell\rangle$.
Then $J=\sum_\ell \mathrm{vec}(M_\ell)\mathrm{vec}(M_\ell)^\dagger$.
By the Choi--Jamio\l{}kowski correspondence, this yields $\mathcal E(\rho)=\sum_\ell M_\ell\rho M_\ell^\dagger$ (Kraus form).
\end{document}
```
